# Svelte + Vite 앱

## 생성하기
```bash
npm create vite@latest
```

## 필요한 패키지 설치하기
```bash
npm install
```

## 개발서버로 실행하기
```bash
npm run dev
```

## tailwindcss 설치 및 초기 세팅
참고링크 : [https://pcm9881.tistory.com/96](https://pcm9881.tistory.com/96)

npm install -D tailwindcss postcss autoprefixer
