# Sroum Admin

## 생성하기

```bash
npm create vite@latest
```

## 필요한 패키지 설치하기

```bash
npm install
```

## 개발서버로 실행하기

```bash
npm run dev
```

## 코드 포맷팅

```bash
npm run format
```
